# /

**Method:** GET

**Description:**
Health check endpoint. Returns API status.

**Response:**
- `status` (string): Service status.
- `service` (string): Service name.

# /ping

**Method:** GET

**Description:**
Simple ping endpoint for monitoring. Returns "ok" and "pong" message.

**Response:**
- `status` (string): "ok"
- `message` (string): "pong"
